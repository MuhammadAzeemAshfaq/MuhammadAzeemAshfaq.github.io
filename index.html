About QA Bug Logger

QA Bug Logger is a Chrome extension designed to help QA engineers, developers, and testers quickly log bugs directly from any web page. It captures bug details like description, screenshot, category, severity, and environment. All bugs are stored in Google Sheets and Google Drive for easy tracking and reporting.

Key Features

  • Capture bugs from any website
  • Automatic screenshot capture
  • Store bug details in Google Sheets
  • Save screenshots to Google Drive
  • Filter bugs by severity, category, browser, and device
