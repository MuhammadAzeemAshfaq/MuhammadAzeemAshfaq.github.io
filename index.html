QA Bug Logger

Welcome to QA Bug Logger

QA Bug Logger is a Chrome extension that allows QA engineers, developers, and testers to quickly log bugs directly from any web page. Capture the bug description, category, severity, device, browser, and screenshots, and save them to Google Sheets and Drive instantly.

Features:

  • Quick bug logging from any web page
  • Automatic screenshot capture
  • Organized bug storage in Google Sheets
  • Drive integration for bug screenshots
  • Filter by category, severity, device, and browser
